# Scraper-Desktop-With-Python-Native-CLI

A simple desktop web scraper built with native Python and Tkinter.

## Deskripsi

Proyek ini adalah aplikasi desktop sederhana untuk scraping data dari halaman web menggunakan Python standar dan antarmuka grafis Tkinter. Cocok untuk keperluan pembelajaran atau automasi pengambilan data ringan.

## Fitur

- Antarmuka GUI berbasis `tkinter`.
- Skrip utama: `scraper.py`.
- Folder `scraped_images/` untuk menyimpan asset yang di-scrape.

## Persyaratan

- Python 3.8+ (direkomendasikan Python 3.10 atau lebih baru)
- Paket: `requests`, `beautifulsoup4` (jika diperlukan oleh `scraper.py`).

Untuk memasang ketergantungan, buat `requirements.txt` berisi contoh berikut lalu jalankan `pip install -r requirements.txt`:

```
requests
beautifulsoup4
```

## Cara Menjalankan

1. Buka terminal di folder proyek:

```
cd "D:\\Python Project List\\Web Scraper Python Native"
```

2. Jalankan aplikasi:

```
python scraper.py
```

## Struktur Proyek

- `scraper.py` — entry point skrip GUI/scraper.
- `scraped_images/` — folder tempat file yang diunduh/disimpan.
